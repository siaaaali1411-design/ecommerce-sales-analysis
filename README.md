# E-commerce Sales Data Analysis

## Project Overview

This project analyzes the Online Retail dataset using Python to uncover sales trends, product performance, and customer value through RFM segmentation.

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Workflow

- Data cleaning
- Feature engineering
- Top-selling product analysis
- Country sales analysis
- Monthly sales trend analysis
- RFM customer segmentation
- Business insights

## Key Findings

- United Kingdom contributed the majority of total sales.
- Sales increased significantly from September to November.
- Champions customers accounted for over 50% of total revenue while representing only a small proportion of customers.
